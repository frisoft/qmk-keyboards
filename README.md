# QMK keyboards

## Use

Initial setup

```bash
make setup
```

Update to the latest QMK

```bash
make setup
```

Flash all keyboards

```bash
make
```

Flash the Dactyl Manuform keyboard

```bash
make dactyl_manuform
```

Clean up

```bash
make clean
```
